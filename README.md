# EXERCISE API

# TC001: 
## Deskripsi
Pengujian dilakukan untuk memverifikasi endpoint API "GET ID 5 - Post Endpoint".
## Langkah-langkah pengujian:
1. Kirim permintaan HTTP GET ke endpoint menggunakan objek pengujian yang telah ditentukan.
2. Verifikasi bahwa respons yang diterima memiliki kode status 200, menunjukkan bahwa permintaan berhasil diproses.
3. Verifikasi bahwa properti 'id' dalam respons sesuai dengan nilai yang diharapkan, yaitu '5', untuk memastikan bahwa entitas yang dikembalikan memiliki ID yang benar.

# TC002: 
## Deskripsi
Pengujian dilakukan untuk memverifikasi endpoint API "GET ID 7 - Photos".
## Langkah-langkah pengujian:
1. Kirim permintaan HTTP GET ke endpoint menggunakan objek pengujian yang telah ditentukan.
2. Verifikasi bahwa respons yang diterima memiliki kode status 200, menunjukkan bahwa permintaan berhasil diproses.
3. Verifikasi bahwa properti 'title' dalam respons sesuai dengan nilai yang diharapkan, yaitu 'quibusdam autem aliquid et et quia', untuk memastikan respons yang benar dikembalikan.
